# Vector Database Directory (`/db/`)

**⚠️ CRITICAL WARNING: DO NOT MODIFY FILES IN THIS FOLDER DIRECTLY.**

This directory contains the physical disk-persisted SQLite and Parquet files managed entirely by **ChromaDB**. 

## Purpose
It stores the high-dimensional mathematical embeddings generated by OpenAI (`text-embedding-3-small`). The RAG pipeline queries this local volume during semantic matching operations.

## How to Interact With this Folder
*   **Resetting the Agent's Brain:** If you face major hallucination issues or corrupt data, delete this entire folder completely and run `python core/ingest.py` from the root directory. ChromaDB will automatically regenerate the folder and mathematically rebuild the node graph from scratch.
*   **File Structure:** The UUID-named folders contain immutable vector chunks. The `chroma.sqlite3` file acts as the indexing ledger tracking chunk boundaries and parent documents.
