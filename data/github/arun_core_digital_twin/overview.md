# Project Overview: ArunCore (AI Digital Twin)

**Repository URL:** [ArunCore Main Repo](https://github.com/neural-arun/ArunCore)

---

## 🚀 The Vision
ArunCore is a **State-of-the-Art Digital Twin** designed to represent Arun Yadav's identity, technical expertise, and project history in a high-fidelity, interactive environment. It serves as an autonomous professional proxy, capable of engaging with recruiters, collaborators, and users while maintaining 100% cognitive alignment with Arun's actual work history.

## 🎯 Primary Objective
The project was born from the need to scale personal branding beyond static resumes and stagnant portfolios. ArunCore acts as a **Retrieval-Augmented Generation (RAG)** gateway that transforms a static knowledge base into a living, breathing AI agent.

## 🌍 Impact & Reach
- **HuggingFace Deployment:** The "Core Brain" lives on HuggingFace Spaces, providing low-latency API access.
- **Vercel Production UI:** A high-end, responsive web interface that works flawlessly on desktop and mobile.
- **24/7 Professional Agency:** Capable of handling hundreds of simultaneous queries, capturing leads, and notifying Arun via Telegram in real-time.

## 🔗 Repository Structure
- **Core Engine:** [ArunCore/core](https://github.com/neural-arun/ArunCore/tree/main/core) - Contains the agent logic, vector DB ingestion, and FastAPI backend.
- **Frontend UI:** [ArunCore/frontend](https://github.com/neural-arun/ArunCore/tree/main/frontend) - A Next.js 15+ application with specialized dual-architecture for mobile/desktop.
- **Data Layer:** [ArunCore/data](https://github.com/neural-arun/ArunCore/tree/main/data) - Semantic knowledge base including detailed project documentation and rules of engagement.

## 🔥 Key Integrations
- **Vector Intelligence:** Powered by ChromaDB and text-embedding-3-small.
- **Reasoning Loop:** Advanced multi-step tool-calling using Groq (Llama-3 & Mixtral models).
- **Communication Hub:** Direct Telegram-to-Phone integration for real-time lead alerts.
