# All Projects Summary: Arun Yadav's Complete Portfolio

**Author:** Arun Yadav
**GitHub Hub:** [Neural Arun](https://github.com/neural-arun)

This document contains a comprehensive, exhaustive list of every active, deployed, and archived agentic AI, data extraction, and automation project built by Arun Yadav.

---

## 1. ArunCore (AI Digital Twin)
- **Description:** A state-of-the-art, retrieval-augmented personal AI agent. It acts as a digital twin for Arun Yadav, synthesizing his professional history, technical expertise, and academic background into an interactive agency proxy.
- **Tech Stack:** Python, FastAPI, Next.js, React, LangChain, ChromaDB, Groq, Cohere Rerank, Telegram API.
- **Status:** Active - Production
- **Repository URL:** [https://github.com/neural-arun/ArunCore](https://github.com/neural-arun/ArunCore)

## 2. Legal RAG System
- **Description:** An 8-stage Retrieval-Augmented Generation pipeline purpose-built for Indian legal documents — IPC, Indian Constitution, and court judgments. Features document-aware structural chunking to eliminate hallucinations.
- **Tech Stack:** Python, LangChain, ChromaDB, Groq, OpenAI Embeddings, PyPDF.
- **Status:** Completed
- **Repository URL:** [https://github.com/neural-arun/legal_RAG_system](https://github.com/neural-arun/legal_RAG_system)

## 3. 99acres Real Estate Scraper Suite
- **Description:** A 3-track web scraping suite for extracting property listing data from 99acres.com. Covers anti-bot bypass, browser automation, and parallel HTTP extraction strategies with CSV output.
- **Tech Stack:** Python, Playwright, HTTPX, ScraperAPI, Pandas, Asyncio.
- **Status:** Completed
- **Repository URL:** [https://github.com/neural-arun/real_state_listing_scraper](https://github.com/neural-arun/real_state_listing_scraper)

## 4. Web Wizard — Playwright Automation Curriculum
- **Description:** A structured, progressive learning repository for advanced web automation and scraping using Playwright for Python — spanning browser automation basics through to distributed, AI-integrated crawler systems.
- **Tech Stack:** Python, Playwright, Pytest, Pandas, PostgreSQL, Redis, Celery, Docker, LangChain.
- **Status:** Work in Progress
- **Repository URL:** [https://github.com/neural-arun/web_wizard](https://github.com/neural-arun/web_wizard)

## 5. Personal AI Digital Twin
- **Description:** A deployed, API-driven AI agent that represents Arun professionally. Answers questions about his background, captures leads, and pushes real-time chat logs to Telegram.
- **Tech Stack:** Python, FastAPI, Uvicorn, Groq, OpenAI SDK, HTTPX, Telegram Bot API.
- **Status:** Deployed
- **Repository URL:** [https://github.com/neural-arun/personal_ai_agent](https://github.com/neural-arun/personal_ai_agent)

## 6. UPPSC PCS 2024 Statistical Audit
