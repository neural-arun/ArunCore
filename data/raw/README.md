# data/raw/

Personal narrative and background context for ArunCore. These files give the agent depth — the human story behind the engineering work that can't be captured in a project README or a LinkedIn profile.

---

## Files

### `personal_background.md`
Arun's personal and academic background in plain, honest prose. Covers:
- Non-linear academic journey: NEET → JEE pivot → self-taught engineering path
- Origin story: started coding on PyDroid 3 on a phone
- Resilience framing: setbacks recontextualised as pivots, not failures
- Core values: pragmatism, project-first learning, ambition
- What drives him: the gap between where he started and where he's going

---

## Purpose
This context is used when the agent is asked personal questions:
- *"How did you get into engineering?"*
- *"What's your background?"*
- *"Did you study CS?"*

Without this file, the agent would either hallucinate or return nothing for these questions. With it, the agent can give an honest, grounded, human response.
